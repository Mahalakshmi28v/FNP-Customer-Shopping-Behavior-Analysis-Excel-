# Data Information

This folder contains the datasets and supporting documents used for the FNP Customer Shopping Behavior Analysis project.

## Files Included
- Excel dataset used for analysis
- Problem statement and business requirements document (PDF)
[Ferns and Petals Sales Analysis.pdf](https://github.com/user-attachments/files/25459632/Ferns.and.Petals.Sales.Analysis.pdf)

## Dataset Description
This project uses three datasets loaded as separate sheets within a single Excel workbook.

### 1. Customers Dataset
Contains customer-level information:
- Customer_ID
- Name
- City
- Contact_Number
- Email
- Gender
- Address

### 2. Products Dataset
Contains product-related details:
- Product_ID
- Product_Name
- Category
- Price (INR)
- Occasion
- Description

### 3. Orders Dataset
Contains transactional order-level information:
- Order_ID
- Customer_ID
- Product_ID
- Quantity
- Order_Date
- Order_Time
- Delivery_Date
- Delivery_Time
- Location
- Occasion
- Revenue

These datasets are connected using common identifiers and analyzed together using Excel.
